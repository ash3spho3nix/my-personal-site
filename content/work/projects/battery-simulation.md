---
title: Battery Simulation Framework
---

A modular platform integrating electrochemical models (DFN, ECM), thermal models, and ageing analysis.  

Designed to move from raw test data → parameter estimation → simulation → analysis in a unified workflow.  

Focus:
- Multi-model integration (ECM + physics-based)
- Scalable architecture
- Open, extensible design
