# 🌐 Personal Website — Vishal Sharma

🔋 Battery Simulation | 🧠 AI Systems | ⚙️ Physics-Driven Engineering

---

## ⚡ Overview

This repository contains the source code for my personal website, built using **Hugo + PaperMod** and deployed on **Netlify**.

The site is not just a portfolio — it’s a structured representation of:

* my work in **battery modeling and simulation**
* systems I’ve built across engineering and AI
* and how I approach problems from a **first-principles perspective**

---

## 🧠 What This Site Represents

Instead of listing everything, the site focuses on three layers:

* 🔋 **Work** → Battery systems, simulation frameworks, engineering tools
* ⚙️ **Systems** → AI infrastructure, code understanding, automation
* 🧠 **Thinking** → How I approach modeling, failure, and system behavior

---

## 🧱 Tech Stack

* **Hugo** — static site generator
* **PaperMod** — lightweight theme
* **Netlify** — deployment

---

## 🚀 Structure

```
content/
├── about/            → Background & narrative
├── projects/         → Selected work
├── thinking/         → How I approach problems
├── ideas/            → Ongoing exploration
├── notes/            → Short-form thoughts
├── experience/       → Professional timeline
```

---

## ⚙️ Running Locally

```bash
# Install Hugo (extended version recommended)
brew install hugo

# Run locally
hugo server -D
```

Open: http://localhost:1313

---

## 🌍 Deployment

The site is deployed via Netlify:

* Build command:

  ```
  hugo --gc --minify
  ```
* Publish directory:

  ```
  public
  ```
